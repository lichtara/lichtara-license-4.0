# Metadata — Lichtara License v4.0

Este diretório contém os metadados oficiais da **Lichtara License v4.0**, incluindo:

- arquivo **CITATION.cff**, utilizado para referência acadêmica e científica;
- arquivo **version-info.json**, contendo a taxonomia de versionamento;
- orientações de citação formal;
- registros essenciais para distribuição em repositórios, indexadores e plataformas públicas.

Todos os metadados seguem o padrão definido nas Seções V e VIII da License e devem ser atualizados sempre que:

- houver nova versão da License,
- forem publicados novos anexos normativos,
- ocorrerem alterações substantivas no ecossistema LICHTARA.
