# Zenodo — Publicação Oficial da Lichtara License v4.0

Este diretório reúne os arquivos utilizados para o **depósito oficial da Lichtara License v4.0 — Structural Living License** no Zenodo.

O Zenodo atua como **repositório institucional de preservação e citação**, enquanto o GitHub mantém a base versionada, operacional e evolutiva do Sistema.

---

## Arquivos incluídos no depósito

* **`metadata.json`**
  Metadados completos da publicação no Zenodo, incluindo autoria, coautoria estrutural, descrição, palavras-chave e referências oficiais.

* **`LICENSE-v4.0.pdf`**
  Documento jurídico consolidado da Lichtara License v4.0, gerado a partir do Documento-Matriz (`master.md`) e anexado ao fluxo de release.

* **`deposit.sh`** *(opcional)*
  Script auxiliar para publicação manual via API do Zenodo, quando aplicável.

## Relação entre GitHub e Zenodo

* **GitHub**
  Mantém o repositório vivo, versionado e auditável do Sistema Lichtara, incluindo o Documento-Matriz (`master.md`), anexos normativos e histórico evolutivo.

* **Zenodo**
  Preserva a **versão formal consolidada** da Licença, garantindo perenidade institucional, rastreabilidade e citabilidade por DOI.

O conteúdo depositado no Zenodo não substitui o repositório GitHub, mas o **ancora juridicamente e academicamente**.

# Normative Annexes — Lichtara License v4

Este diretório contém os **anexos normativos oficiais** que integram a Lichtara License v4 — Structural Living License.

Os anexos são parte **indissociável** da Licença e regulam, em profundidade:

* princípios ético-regenerativos,
* linguagem de conformidade vibracional,
* modelos de autoria e coautoria,
* mecanismos de impacto,
* estruturas de governança e certificação.

Os anexos devem ser interpretados **em leitura combinada** com o Documento-Matriz (`master.md`), que constitui o núcleo normativo da Licença.
