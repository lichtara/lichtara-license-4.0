# Zenodo — Publicação Oficial da Lichtara License v4.0

Este diretório contém os arquivos utilizados para o depósito da Lichtara License v4.0 no Zenodo.

## Arquivos

- `metadata.json` — Metadados completos da publicação.
- `LICENSE-v4.0.pdf` — Gerado automaticamente pelo CI/CD e anexado no fluxo de release.
- `deposit.sh` (opcional) — Script para publicação manual via API.
