# Lichtara License v4.0 — Documento Oficial

A **Lichtara License v4.0** é o instrumento jurídico-ético-vibracional que regula o uso, implementação, derivação, certificação e conformidade da Obra LICHTARA enquanto sistema vivo de conhecimento, tecnologia e consciência.

Este repositório contém:

- o texto oficial consolidado (LICENSE-v4.0.md),
- o master para geração do PDF institucional (master.md),
- o template LaTeX oficial,
- os anexos administrativos e normativos,
- os fluxos públicos e diagramas,
- os short-forms para usos específicos,
- e os arquivos de metadados para o Zenodo.

A versão PDF é gerada via Pandoc + XeLaTeX com o template institucional localizado em `/pdf/template.tex`.

DOI oficial: **10.5281/zenodo.17844329**

Publicação institucional:  
https://github.com/lichtara/lichtara-license-v4
